**
** Module DOC.
** Notice explicative pour l'installation des outils complémentaires.
**
**
** Testé sous Debian Etch.
**

L'outil d'indexation intégré nécessite l'installation préalable d'utilitaires sur le serveur.

** Installation de catdoc, catppt, xls2csv, pdfinfo, pdftotext, unrtf, jhead, html2text, gd
- catdoc permet d'extraire le contenu des documents MS Word
- catppt permet d'extraire le contenu des documents MS Powerpoint
- xls2csv permet d'extraire le contenu des documents MS Excel
- pdftotext permet d'extraire le contenu des documents Adobe PDF
- html2text permet d'extraire le contenu des documents HTML
- pdfinfo permet d'extraire les métadonnées des documents Adobe PDF
- jhead permet d'extraire les métadonnées des documents JPG
- gd permet de générer des miniatures d'images

> apt-get install catdoc xpdf-utils unrtf jhead html2text php5-gd

** Installation de hachoir (extraction de metadonnées) :

Ajouter le dépôt non-free de Debian dans /etc/apt/sources.list (pour python-profiler utilisé par hachoir)
> deb http://ftp.fr.debian.org/debian/ etch main contrib non-free

Installer python-profiler.
> apt-get install python-profiler

Ajouter les dépots suivants (hachoir n'est pas encore disponible dans la version stable de debian mais dans la unstable uniquement) dans votre sources.list : 
> deb http://plumbear.free.fr/debian etch main
> deb-src http://plumbear.free.fr/debian etch main

Récupérer la clé publique.
> wget -q http://plumbear.free.fr/BC663996EE77C6DF.pubkey -O- | apt-key add -

Installer hachoir-metadata.
> apt-get install python-hachoir-metadata
