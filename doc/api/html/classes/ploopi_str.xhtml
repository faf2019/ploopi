  • is_url() — Vérifie qu'une url est valide

  • make_links() — Rend les liens d'un texte cliquables

  • utf8encode() — Encode une chaîne en UTF8
  • xmlentities() — Encode les caractères spéciaux d'une chaîne pour qu'elle puisse être intégrée dans un document XML
